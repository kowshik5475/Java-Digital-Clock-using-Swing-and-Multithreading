# Java Digital Clock

A simple digital clock application built in Java using Swing and multithreading.

## Features
- Displays current time in hh:mm:ss format
- Updates every second
- Uses Java Swing for GUI and a separate thread to refresh the time

## Technologies Used
- Java
- Swing (JFrame, JButton)
- java.util.Calendar
- java.text.SimpleDateFormat

## How to Run
1. Compile the program:
   javac DigitalWatch.java
2. Run the program:
   java DigitalWatch
